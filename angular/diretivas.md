# Diretivas

São uma forma de passar instruções para o template do código html dando bastante poder ao código.

Existem as ***diretivas estruturais*** que interagem com a view e modificam a estrutura DOM e/ ou código HTML.
Exemplos:
  - <b style="color: #12Af16"> NgIf </b> - cria uma condicional para a visualização do elemento_. 
  - <b style="color: #12Af16"> NgFor </b> - Cria um laço de repetição iterando com cada item da lista_.
  - <b style="color: #12Af16"> NgSwitch </b>- _Cria um conjunto de diretrizes que alternam de acordo com as alternativas_. 

Existem também as ***diretivas de atributos*** que interagem com o elemento em que foram aplicadas e não alteram o DOM.
Exemplos:
   - <b style="color: #12Af16"> NgClass </b>- _Adiciona e remove um set de uma classe CSS_.
   - <b style="color: #12Af16"> NgStyle </b>- _Adiciona e remove um set de estilo HTML_.
   - <b style="color: #12Af16"> NgModel </b>- _Adiciona o two-way data binding para um elemento do formulário HTML_.
